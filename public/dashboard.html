📡 Endpoints
POST
/api/orders
Create a new payment order. Returns order_id and payment_url.
GET
/api/orders/:orderId
Check order status. Poll this or use webhook for real-time updates.
GET
/api/orders
List all orders with stats (revenue, paid, pending, failed).
POST
/webhook/payment
Webhook receiver. Gateway calls this when payment is confirmed. HMAC-SHA256 verified.
